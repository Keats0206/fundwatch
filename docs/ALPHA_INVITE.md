## Quick Demo Script

1. **Log in** with your Mantis Ventures credentials.
2. **Portfolio** — Watch input at top: type a company name (e.g. Rogo, Jeeves).
3. **Click Watch** — If company exists: you’re taken to the company page and an alert runs. If not: Add Company modal opens.
4. **On a company page** — Click **Run alert** → wait for search + brief → read **What You Can Do** and **Why It Matters**.
5. **Sources** — Expand “Sources” to see where the info came from.
6. **Brief history** — Use the chevrons to browse previous briefs if you’ve run alerts weekly.
